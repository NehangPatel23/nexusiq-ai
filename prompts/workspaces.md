# Prompt: Workspaces

**Version:** 1.0.0  
**Feature:** workspaces  
**Purpose:** Workspace CRUD within organization

## Inputs
- Workspace name, description
- Organization context

## Outputs
- Workspace with unique slug per organization

## Constraints
- Member+ can create
- Admin+ can edit
- Admin+ can delete, restore, and permanently delete
- Slug auto-generated, editable (admin+)
